# DailyTask 使用说明与 UAT 测试清单

这份文档给家长使用。前半部分说明“怎么用”，后半部分可以当作 UAT 测试清单，一项一项打勾，看看这个网页是否符合家庭实际需求。

## 1. 这个网页是做什么的

DailyTask 是一个给孩子用的任务规划网页。

它的重点不是让孩子每天完成所有任务，而是帮助孩子学会两件事：

- 先把事情分分类。
- 再从分类好的事情里，选择今天真的要做的少量任务。

所以请记住：

**放进四象限，不等于今天必须做。**

只有任务卡上的星星变成 `★`，才表示“今天要做”。

## 3. 数据保存在哪里

当前版本没有账号、没有后端、没有云同步。

任务数据保存在当前浏览器本机的 IndexedDB 里。

也就是说：

- 电脑浏览器里的数据，只在这台电脑这个浏览器里。
- 平板浏览器里的数据，只在这台平板这个浏览器里。
- 手机、平板、电脑之间不会自动同步。
- 清理浏览器网站数据后，任务数据可能会消失。

这对早期试用是好事：不用担心孩子的数据被上传到服务器。

## 4. 主页面区域说明

桌面端大致分成左右两边：

- 左边：四象限区域。
- 右边：新增任务、常用任务、今日计划。

手机和平板上会变成上下排列。

### 4.1 四象限

四个象限分别是：

- `🔥 いま やる`
  - 现在比较重要、比较需要优先处理的事情。
- `🌱 そだてる`
  - 需要慢慢积累的成长类事情。
- `🚗 とつぜん`
  - 临时出现、突然插入的事情。
- `🍬 おたのしみ`
  - 娱乐、休闲、有空再做的事情。

任务可以在四个象限之间拖来拖去。

### 4.2 常用任务

第一次打开时，会自动出现 8 个常用任务：

- `かんあぷり`
- `えいごれんしゅう`
- `こくごれんしゅう`
- `すとれっち`
- `おえかき`
- `えほん`
- `かんじ`
- `うくれれ`

这些任务可以拖到左侧四象限里。

同一个常用任务，同一天只能放进四象限一次。

### 4.3 新增任务

在 `あたらしい たすく` 区域可以新增任务。

建议输入孩子能读懂的平假名。

新增后，任务会出现在 `いつもの たすく` 常用任务区。

### 4.4 今天做的星星

四象限里的每张任务卡都有星星按钮：

- `☆ きょう やる`
  - 还没有加入今日计划。
- `★ きょう やるよ`
  - 会加入今日计划。

重要规则：

**只有点亮星星的任务，才会进入今日计划。**

任务移动到别的象限后，星星状态会保留。

### 4.5 生成今日计划

点右侧按钮：

```text
よていを きめる
```

系统会把所有 `★` 的任务加入今日计划。

四个象限里的任务，只要点亮星星，都可以进入今日计划。

今日计划会按原象限分组显示：

- `🔥 いま やる`
- `🌱 そだてる`
- `🚗 とつぜん`
- `🍬 おたのしみ`

如果没有点亮任何任务，会提示：

```text
きょう やることを えらんでね
```
如果想从今日计划里删掉某个任务的话，需要从象限里先消灭星星，然后再点击よていをきめる

### 4.6 完成打卡

今日计划里的任务有 `できた` 按钮。

点击后：

- 任务状态会变成完成。
- 完成次数会记录。
- 连续完成次数会更新。

同一个任务同一天重复点击，不应该重复增加完成次数。

### 4.7 家长设置

右上角有齿轮按钮 `⚙`。

需要长按进入家长设置。

家长设置里可以：

- 清空今天。
- 清空全部数据。
- 编辑任务时设置频率：
  - 每天型
  - 每周次数型
  - 自由型

### 4.8 频率设置和轮换建议

任务的 `なおす` 里面有 `どれくらい` 设置。

这个设置不是强制规则，不会自动把任务加入今日计划，也不会禁止孩子继续选择任务。

它的作用是：当孩子已经做得足够多时，系统给一个温和建议，让孩子考虑换一个任务。

频率有三种：

- `まいにち`
  - 每天型。
  - 适合每天都希望做一点的任务。
  - 如果连续完成达到 4 次，再把它从 `☆` 点成 `★`，或点击 `よていを きめる` 准备加入计划时，会提示是否换一个。

- `しゅうに Nかい`
  - 每周次数型。
  - 适合一周做几次就够的任务。
  - `N` 可以设置为 1 到 7。
  - 如果本周完成次数已经达到设定次数，再选择它时，会提示这个任务本周已经做够了，可以考虑换一个。

- `じゆう`
  - 自由型。
  - 适合兴趣、娱乐、偶尔做的任务。
  - 不触发轮换建议。

轮换建议弹出时：

- 选择继续，就保持 `★`，任务仍然可以进入今日计划。
- 选择换一个，就把该任务恢复成 `☆`。

例子：

```text
かんあぷり 设为 しゅうに 3かい。
如果这周已经完成 3 次，再点亮它，
系统会建议“这周已经做够了，要不要选别的？”
但仍然允许继续选择。
```

## 5. 推荐家庭使用流程

每天可以按这个顺序使用：

1. 和孩子一起看 `いつもの たすく`。
2. 把今天想到的任务拖进四象限。
3. 先只分类，不急着说“都要做”。
4. 分类完成后，让孩子自己点亮几个 `★`。
5. 点击 `よていを きめる`。
6. 和孩子确认今天的计划。
7. 做完一项，就让孩子自己点 `できた`。

家长提醒时，建议说：

- “这个先放到哪里呢？”
- “今天真的要做的是哪几个？”
- “糖果区没有做完也没关系。”
- “点亮星星的才是今天的计划。”

尽量避免说：

- “这里面的都要完成。”
- “你放进去的怎么没做？”
- “今天必须全部打卡。”
